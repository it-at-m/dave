# SVG-Grafiken

Die SVG-Grafiken zur Darstellung der Belastungspläne Qjs, Fjs und Qu wurden mit Inkscape erstellt.

Die Grafiken wurden z.T. manuell nachbearbeitet, da Inkscape bestimmte Einstellungen nicht unterstützt.
Folgende Attribute müssen manuell in einem Texteditor bearbeitet werden:
  * -inkscape-font-specification aus Plain SVG entfernen
  * transforms für Verschiebungen entfernen (stattdessen nur mit Koordinaten arbeiten)
